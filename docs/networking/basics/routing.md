### 2. –í–Ω–µ—à–Ω–∏–π –ø—Ä–æ—Ç–æ–∫–æ–ª —à–ª—é–∑–∞ (EGP - Exterior Gateway Protocol)

BGP –æ–ø–µ—Ä–∏—Ä—É–µ—Ç –Ω–µ –ø—Ä–æ—Å—Ç–æ –º–µ—Ç—Ä–∏–∫–∞–º–∏, –∞ —Å–ª–æ–∂–Ω—ã–º–∏ –ø–æ–ª–∏—Ç–∏–∫–∞–º–∏, –æ—Å–Ω–æ–≤–∞–Ω–Ω—ã–º–∏ –Ω–∞ –±–∏–∑–Ω–µ—Å-–¥–æ–≥–æ–≤–æ—Ä–µ–Ω–Ω–æ—Å—Ç—è—Ö –º–µ–∂–¥—É –ø—Ä–æ–≤–∞–π–¥–µ—Ä–∞–º–∏.
